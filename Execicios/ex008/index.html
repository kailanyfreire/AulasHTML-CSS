Principais Formatações de Textos

Negrito / Destaque

Exitem duas formas de usar o negrito: com a tag <strong> (com semântica, ou seja, com significado de que o texto destacado é forte, é importante) e com a tag <b> (sem semântica, ou seja, sem significado especial, só quero deixar em negrito).

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i> (sem semântica).

Nessa frase temos um termo com ênfase usando a tag <em> (com semântica, ou seja, com significado de que o texto destacado tem ênfase).

Texto Marcado

Podemos marcar um texto como se estivessemos usando um marca texto usando a tag <mark>.

Texto grande e pequeno

Nessa frase temos um texto grande (com a tag <big>, que está obsoleta) e um texto pequeno (com <small>).

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois. També existe a tag <u> para sublinhar textos (sem semântica).
